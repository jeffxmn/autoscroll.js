Including delayedscroll.js in your website

delayedscroll.js is a JavaScript library that enables delayed, automatic, scrolling on one-page, full-width websites.

The library is lightweight and implementation is straightforward.

To include the library in your website, you need to include two files:

<link href="css/delayedscroll.css" rel="stylesheet">

and:

<script src="js/delayedscroll.js"></script>

Note: delayedscroll.js also depends on JQuery. Make sure you have JQuery included!

Using delayedscroll.js

To create full-height-width pages with the delayed scrolling effect, you must first add a delayedscroll class to a container <div> directly under <body>:

<body>
    <div class="delayedscroll">
        <!-- Pages here -->
    </div>
</body>

Each page is then a nested <div> under the delayedscroll div, labeled as page-*, where * is the page number, starting from 1:

<body>
    <div class="delayedscroll">
        <div class="page-1">
            <span>Hi, I'm Page 1</span>
        </div>
        <div class="page-2">
            <span>Nice meeting you! I'm Page 2</span>
        </div>
        <div class="page-3">
            <!-- AFK -->
        </div>
    </div>
</body>

And that's it! You should now be experiencing a surreal delayed scrolling effect on full-height-width pages. Enjoy! :)

Browser Support

delayedscroll.js has been tested on:

- Chrome 25.0 and up
- Firefox 20.0.1 and up
- Internet Explorer 10
